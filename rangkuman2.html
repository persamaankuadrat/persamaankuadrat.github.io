B. Menyelesaikan Persamaan Kuadrat

Rangkuman

Cara-cara menyelesaikan persamaan kuadrat adalah:

  1. Mempaktorkan

    2.     1. Jika \[x^2 + bx + c = (x + p)(x + q) = 0\] (dengan \[b = p + q\] dan \[c = p . q\]), maka : \[x + p = 0\] atau \[x + q = 0\]

        2. Jika \[ax^2 + bx + c =\frac {1}{a} (ax + p)(ax + q) = 0\] (dengan \[b = p + q\] dan \[c = p . q\]), maka : \[x + p = 0\] atau
           \[x + q = 0\]

        3. Jika \[p \cdot q = 0\], maka \[p = 0\] atau \[q = 0\] atau \[p = q = 0\]

  2. Melengkapkan kuadrat sempurna (syarat \[a = 1\])

    3.       Jika \[x^2 + bx + c = 0\], maka :
          \[x^2 + bx + \left ( \frac{b}{2} \right )^{2} = -c + \left ( \frac{b}{2} \right )^{2}\Leftrightarrow \left (x + \frac{b}{2} \right )^{2} = -c +\left ( \frac{b}{2} \right )^{2}\]

  3. Rumus Kuadratis (rumus abc)

    4.       \[x = \frac{-b \pm \sqrt{b^{2} - 4 ac}}{2a}\]
