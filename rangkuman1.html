A. Pengertian Persamaan Kuadrat

Rangkuman

  1. Persamaan kuadrat disebut juga persamaan pangkat dua.
  2. Pengertian persamaan kuadrat:
    Persamaan kuadrat merupakan salah satu persamaan matematika dari variabel yang mempunyai pangkat tertinggi dua, dengan x merupakan variabel a, b, merupakan koefisien dan c merupakan konstanta.
  3. Bentuk umum persamaan kuadrat adalah:
                   ax2 + bx + c = 0, dengan a ≠ 0
  4. Bentuk ax2 + bx = 0 disebut persamaan kuadrat yang tidak lengkap. Bentuk ax2 + c = 0 disebut persamaan kuadrat sempurna.
